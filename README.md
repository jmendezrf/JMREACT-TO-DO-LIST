# JMREACT TODO LIST

JMREACT is a React sample project for manage to-dos, saving the states on the browser localstorage.

## Installation

clone the repository

```bash
git clone <url> 
```

c
```bash
npm install 
```

## Usage

To run the project
```react
 npm start
```
enjoy :)
